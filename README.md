## 🌟 Overview

**AlgoFlow Visualizer** is an interactive web platform designed to make **Data Structures & Algorithms intuitive and visual**.

Instead of memorizing code, users can:

* 👀 *See algorithms in action*
* ⚡ *Understand step-by-step execution*
* 🧠 *Reinforce learning through quizzes*

## ✨ Features

### 🎯 Algorithm Visualizations

✔ Step-by-step animations

✔ Adjustable speed control

✔ Color-coded operations (compare, swap, sorted)

✔ Pseudocode + memory tricks 

✔ Time & space complexity

Supported algorithms:

* 🔵 Bubble Sort
* 🟣 Selection Sort
* 🟢 Insertion Sort
* 🟡 Merge Sort
* 🔴 Quick Sort
* 🟠 Heap Sort
* 🔍 Linear Search
* ⚡ Binary Search

---

### 📊 Comparison Dashboard

* Run multiple algorithms simultaneously
* Compare:

  * Number of comparisons
  * Number of swaps
* Real-time performance insights

---

### 🧠 Quiz System

* Topic-based quizzes
* Mixed practice tests
* Instant scoring system
* Interactive UI

---

### 🎨 UI/UX Highlights

* 🌙 Dark theme
