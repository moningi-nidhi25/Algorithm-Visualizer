Algorithms

Sorting

Bubble Sort

Repeatedly swaps adjacent elements if they are in the wrong order.

Time: O(n²) | Space: O(1)
Sorting

Selection Sort

Selects the minimum element from the unsorted part and places it at the beginning.

Time: O(n²) | Space: O(1)
Sorting

Insertion Sort

Builds a sorted array one element at a time by inserting elements into their correct position.

Time: O(n²) | Space: O(1)
Sorting

Heap Sort

Uses a binary heap to repeatedly extract the maximum (or minimum) element.

Time: O(n log n) | Space: O(1)
Sorting

Quick Sort

Divides the array using a pivot and recursively sorts the partitions.

Time: O(n log n) avg | O(n²) worst
Sorting

Merge Sort

Divides the array into halves, sorts them, and merges them back together.

Time: O(n log n) | Space: O(n)
Searching

Linear Search

Checks each element sequentially until the target is found.

Time: O(n) | Space: O(1)
Searching

Binary Search

Repeatedly divides a sorted array to find the target efficiently.

Time: O(log n) | Space: O(1)
