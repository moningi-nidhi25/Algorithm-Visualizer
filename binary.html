🧠 Pseudocode

low = 0, high = n-1
while low ≤ high:
  mid = (low+high)/2
  if (arr[mid] == target) → found
  else if (target < arr[mid]) → high = mid-1
  else → low = mid+1

💡 Memory Trick

✂️ Divide the array in half

Discard one half each step
