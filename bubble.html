Bubble Sort Visualizer

Speed: Medium

🎨Legend

■ Normal (Unsorted)

■ Comparing

■ Swapped

■ Sorted

⏱ Time Complexity

🟢 Best: O(n)

🟡 Average: O(n²)

🔴 Worst: O(n²)

💾 Memory Usage

📦 Extra Space: O(1)

⚡ In-place sorting

📌 When to Use

✔ Small datasets

✔ Nearly sorted arrays

✖ Large datasets

🧠 Pseudocode

for i = 0 to n-1:
  for j = 0 to n-i-2:
    if arr[j] > arr[j+1]:
      swap(arr[j], arr[j+1])

💡 Memory Trick

🫧 “Bubbles rise to the top”

Largest elements move to the end
